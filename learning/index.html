Educating for the Future

IEEE Learning aims to teach students valuable skills to use in their careers that are often bypassed in Purdue's curriculum. Many workshops are held throughout the year, and some past examples are Code Cafe (Python), Arduino, soldering, Git, EAGLE, SOLIDWORKS, and many others!

The IEEE Learning team is always looking for driven students to help plan workshops by educating, writing curriculum, and planning events! If interested, contact Sebastian Gruenwald through the IEEE Slack workspace.

Eagle Session 1 - Schematics

Presentation
Written Guide

Learn how to create a schematic for a project in Eagle. This tutorial covers adding resistors, capacitors, transistors, and integrated circuits (ICs) to a 4-bit adder.

Eagle Session 2 - Layout & Routing

Presentation
Written Guide

Learn how to place components for a board effectively and route them. Lots of considerations such a noise, current, manufacturability, and solder-ability.

Eagle Session 3 - Custom Components

Presentation
Written Guide

Learn how to create custom components for specialized ICs or connectors in your board.
