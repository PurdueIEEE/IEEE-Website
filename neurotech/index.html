Neurotech Committee

The Purdue IEEE Neurotech Committee (NTC) works on creating Brain Machine Interfaces using Machine Learning and Electroencephalogram technology. Designing signal processing networks, programming neural networks and conducting human testing are all core aspects of the NTC.
